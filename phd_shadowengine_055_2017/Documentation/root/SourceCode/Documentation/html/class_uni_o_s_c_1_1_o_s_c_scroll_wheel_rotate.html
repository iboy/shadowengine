Detailed Description

This class is a blueprint for your own implementations of the abstract class UniOSCEventDispatcher Dispatcher forces a OSCConnection to send a OSC Message. //Don't forget the base callings !!!!

The data sent is the movement of a single controller in the following index order data at index [0] int isDragging data at index [1] float x data at index [2] float y data at index [3] int isPinching data at index [4] float z scale data at index [5] int isTwisting data at index [6] float z rotation (euler) data at index [7] int IsDoubleTapped
